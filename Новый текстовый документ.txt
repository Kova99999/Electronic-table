Электронная ведомость
Должно быть два типа пользователей: преподаватель (может редактировать)
и студент (может только просматривать)
Можно иметь несколько ведомостей под разные предметы
Можно динамически задавать количество строк и столбцов
На оценку 5:
проект должен лежать на github
проект должен собираться либо с помощью Maven, либо с помощью Ant (компиляция, запуск тестов, упаковка тестов, очистка)
